# RSE Systems Website

Offizielle Website für [RSE Systems](https://rse-systems.github.io) – gehostet mit GitHub Pages.

## Struktur
- `index.html` – Startseite
- `kontakt.html`, `impressum.html`, `datenschutz.html`
- `assets/` – CSS, JS, Bilder

## Deployment
Die Seite wird automatisch über `main` bei GitHub Pages bereitgestellt.
